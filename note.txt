===========Create react project with Vite=== https://vitejs.dev/guide/
- create folder "portfolio"
- npm create vite@latest . --template react
- Select react
- select Javascript
- npm install
- npm run dev
- Select http://localhost:5173/

========Add tailwind css====  https://tailwindcss.com/docs/guides/vite
- npm install -D tailwindcss postcss autoprefixer
- npx tailwindcss init -p
- add this coe to "tailwind.config.js"
    content: ["./index.html", "./src/**/*.{js,ts,jsx,tsx}", ],
- npm run dev
==========To change image to icon=======
